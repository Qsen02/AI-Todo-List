# Todo list
This is AI generated web app using Codex with GPT-5.5 model. This is simple app for creating and reading notes.
# Technologies
React with Typescript
# Features
- You can create notes.
- Edit notes.
- Delete notes.
- Toggle light and dark theme.
# How to start app
Open terminal and run command `npm run dev`, after this you must see a link in terminal. Open the link and you must see the app.
